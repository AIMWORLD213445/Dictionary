# _Dictionary_

#### _September, 16th, 2016_

#### By _**Brian Dutz**_

## Description

_A web application that allows users to generate a dictionary by adding their own words and definitions of those words_

## Setup/Installation Requirements

_Clone the repo and open it up_

## Specifications

* Behavior: Store a definition
  * **Example input:** the sound a dog makes
  * **Example output:** the sound a dog makes
* Behavior: Correctly assign the stored definition to a stored word
  * **Example input:** word:bark, definition:sound a dog makes
  * **Example output:** word:bark, definition:sound a dog makes
